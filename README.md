
# 🌙 Luna AI - Analytics & Relatórios Inteligentes


A **Luna AI** é uma aplicação de análise de dados automatizada que utiliza Inteligência Artificial para transformar planilhas de vendas em relatórios estratégicos e insights de negócios em segundos.

---

## 🚀 Funcionalidades

- **Processamento de Dados:** Carregamento de arquivos CSV com detecção automática de codificação.
- **Visualização Gráfica:** Geração de gráficos dinâmicos de faturamento por categoria usando Matplotlib.
- **Inteligência Artificial:** Integração com o modelo **Llama 3.3 (Groq)** para análise executiva e recomendações estratégicas.
- **Exportação em PDF:** Gera um relatório profissional contendo os gráficos e a análise da IA para download imediato.
- **Modo Demo:** Permite testar a ferramenta instantaneamente com dados fictícios.

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Python 3.13
* **Interface:** Streamlit
* **IA:** Groq API (Llama 3.3 70b)
* **Análise de Dados:** Pandas
* **Gráficos:** Matplotlib
* **PDF:** FPDF

## ⚙️ Como rodar o projeto localmente

1. Clone o repositório:
   
git clone [https://github.com/Dedeinha81/Automa--o_relatorio_ia.git]

   ---
   
2. Instale as dependências:

pip install -r requirements.txt

Configure sua chave da API Groq no código ou como variável de ambiente.

---

3. Execute o app:

streamlit run app.py
